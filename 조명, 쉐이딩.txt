< 물체의 재질과 표면의 방향 >
- 광원으로부터 그 지점을 잇는 벡터가 법선벡터와 어떤 각도로 놓여 있는지에 따라 빛의 양이 달라짐

[광원]
: 직접적 광원 = 점광원, 지향광원, 무대광원
: 간접적 광원 = 주변광원

점광원 
: 광원 P0에서 도착지점 P로의 빛의 세기 I(P, P0)은 1/(P-P0)의 노름 * I(P0)이다.
: 점광원은 주그림자 영역 하나만 생김. 
: 점광원에서는 그 위치가 주요한 요소이다.

지향광원
: OpenGL에서는 4차원으로 표현할 때 네 번째 값이 0이면 지향광원, 1이면 점광원이다.
: 광원의 기하학적 특징을 나타내는 (x,y,z)는 점광원의 경우에는 광원의 위치를 나타내지만
지향광원의 경우에는 진행하는 빛의 방향벡터를 나타낸다.

무대광원
: 중심방향에 가까운 빛은 세기가 강하고 중심에서 멀어질수록 빛의 세기가 약해지다가
특정각도를 벗어나면 빛이 사라짐

주변광원
: 공간 속에 있는 물체들의 표면에서 반사되는 빛에 의해서 이루어지는 광원
: 따라서 광원의 위치, 광원의 방향은 의미가 없음
: 공간의 모든 위치에는 동일한 세기의 주변광이 도달한다.

< 광원과 물체와의 상호작용 >
: 물체의 표면에 입사된 빛의 에너지는 표면에서 반사, 굴절, 흡수됨
: 물체 표면과 빛의 상호작용 3가지는 완전 정반사면, 완전 굴절면, 완전 난반사면이 있음
: 대부분의 물체의 지점에 들어오는 입사 광선은 정반사와 난반사 발생.
: 굴절은 유리와 같은 특별한 종류일 때만 발생함.

완전 정반사면
: 하나의 입사 광선은 표면에서 입사각과 같은 크기의 반사각을 가지는 하나의 반사 광선을 가짐

완전 난반사면
: 하나의 입사 광선이 물체 표면에 들어가 모든 방향으로 동일한 밝기의 반사광을 반사시키는 물체의 표면
: 카메라의 위치가 바뀌어도 그 지점으로부터 반사되는 난반사의 밝기는 동일함

완전 굴절면
: 하나의 입사광선이 물체 표면에 들어가 
물체를 뚫고 물체 재질이 가지는 특성에 따른 굴절각도로 하나의 반사광선을 가지는 표면

<퐁의 반사 모델>
: 공간 속의 빛은 무수하기에 하나의 광원에서 물체의 임의의 정점에 도달하는 빛은 하나라고 가정하고
그 빛이 반사되는 것을 3가지로 분류하여 계산하는 방법
: 간접적인 광원에 대한 조명 계산법은 주변광원으로 대신해서 사용함.
: 물체 표면의 한 지점에서 반사되는 빛의 밝기는 정반사, 난반사, 주변 반사의 합으로 표현됨

난반사(diffuse)
: 물체면과 광원과의 공간적인 관계에 따라 빛의 세기가 달라짐.

정반사(specular)
: 매끈한 표면으로 반사가 되어 하이라트 생성됨.
: 시선벡터와 방향벡터가 필요함

주변 반사(ambient)
: 시점의 위치에 무관하게 주변에서 들어오는 빛을 모든 방향에 동일한 세기로 반사함.

자체 발광(emissive)
: 표면 자체에서 발광하는 빛의 양을 뜻함.

위의 4가지 반사를 모두 더하여 조명값을 처리함.

[ 조명 활성화 ]
: glEnable(GL_LIGHTING);
: glEnable(GL_LIGHT i ); i = 0~7
광원 매개 변수 반환 ==> glLightfv( ) : glLightfv(광원, 지정할 속성 이름, 속성값)
(속성값 종류)
: GL_AMBIENT : 광원의 주변광
: GL_DIFFUSE : 광원의 난반사
: GL_SPECULAR : 광원의 정반사


< 다면체 쉐이딩 >
폴리건 쉐이딩
: 대부분의 물체가 메쉬 형태의 다면체 형태로 이루어짐
: 부드러운 곡면도 렌더링 할 때 다면체 형태로 전환하여 사용됨
: 계산량에 따라서 플랫 쉐이딩, 그라운드 쉐이딩, 퐁 쉐이딩으로 나뉨

플랫 쉐이딩
: 가장 간단한 쉐이딩 방법으로 빠른 시간에 처리가능
: 동일한 면에 놓여있는 모든 점에 동일한 색상 부여
: 매치 밴드 효과 발생=이웃하는 면 사이에 선분이 그려진 것처럼 보임
: 실행시간 빠름
: glShadeModel(GL_FLAT);

구로드 쉐이딩
: 정점의 법선벡터는 인접면의 법선벡터를 평균하여 구함.
: 각 정점에서 조명값이 계산되고 계산된 색상이 표면 전체로 보간됨
: 정점에서의 법선벡터를 이웃한 면의 법선벡터들의 평균으로 구함
: 정점에서의 빛의 세기를 퐁 반사모델을 통해서 계산함.
: 내부의 점에서는 법선벡터가 계산되지 않아서 내부에서는 하이라이트 표현 불가능함.
: 매치 밴드 효과 발생
: glShadeModel(GL_SMOOTH);

퐁 쉐이딩
: 정점의 법선벡터는 인접면의 법선벡터를 평균하여 구함.
: 각 정점의 법선벡터를 이용하여 면에 해당하는 각 픽셀의 법선벡터를 보간함
: 그라운드 쉐이딩보다 느림

[ 쉐이딩 모델 ]
: glShadeModel(GL_SMOOTH), glShadeModel(GL_FLAT);

[물체 표면 종류]
: glMaterialfv(face, pname, *params);
: face 종류 : GL_FRONT, GL_BACK, GL_FRONT_AND_BACK
: pname 종류 : GL_AMBIENT(주변광에 대한 물체색),
	     :  GL_DIFFUSE(난반사에 대한 물체색), 
	     : GL_AMBIENT_AND_DIFFUSE(주변광+난반사에 대한 물체색)
	     : GL_SPECULAR(정반사에 대한 물체색)
	     : GLEMISSION(물체의 발광색)
 	     : GL_SHININESS(정반사의 광택계수)

< 전역조명 모델, 지역 조명 모델 >
Global Illumination Model(전역 조명모델) : 다른 물체면에서 반사되어 입사되는 빛까지 고려한 조명모델
Local Illumination Model(지역 조명모델) : 광원으로부터 직접 물체면으로 입사되는 빛만 고려한 조명모델
